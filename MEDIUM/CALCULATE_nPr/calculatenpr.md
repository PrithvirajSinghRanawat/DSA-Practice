# PROGRAM TO CALCULATE VALUE OF <sup>n</sup>P<sub>r</sub>.

*Given two numbers, n and r, the task is to compute nPr, which represents the number of ways to arrange r elements from a set of n elements. It is calculated using the formula n!/(nâˆ’r)!, where "!" denotes the factorial operation.*

**nPr = n! / (n - r)!**

           r = 2
